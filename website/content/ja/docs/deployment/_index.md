Manael はステートレスな Docker コンテナとして配布されており、さまざまな実行環境に導入できます。このセクションではデプロイ先ごとにページを分け、必要な情報へ直接たどれるようにしています。

利用する環境に応じて次のガイドを参照してください。

- [Docker Compose](docker-compose/) はローカル開発や単一ホストでの簡単な構成向けです。
- [Google Cloud Run](cloud-run/) は自動スケールするマネージドなサーバーレス環境向けです。
- [Kubernetes](kubernetes/) はレプリカ数やリソース制限を細かく制御したい場合に向いています。

本番環境の運用要件については、あわせて [プロダクションガイド](/ja/docs/production-guide/) も参照してください。
