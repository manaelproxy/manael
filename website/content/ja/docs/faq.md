## Manael はなぜ PNG を AVIF に変換しないのですか? {#png-avif}

PNG から AVIF へのエンコードは、互換性・エンコード速度・アルファチャンネルの扱いを優先するために意図的に無効化されています。PNG はクライアントが対応している場合、引き続き WebP へ変換されます。

詳細は[対応フォーマット]({{< relref "supported-formats" >}})ページの変換対応表をご覧ください。

## Amazon S3 バケットの URL を `-upstream_url` として使用できますか? {#s3-upstream}

はい、S3 バケットがパブリックにアクセス可能か、Manael が稼働しているネットワークからアクセスできる状態であれば使用できます。

## Manael は変換した画像をキャッシュしますか? {#caching}

いいえ。Manael は完全にステートレスな処理プロキシとして動作します。キャッシュには Manael の前段に CDN を配置してご利用ください。
