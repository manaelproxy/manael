# Security Policy

## Supported Versions

Manael currently supports the following versions for security updates:

| Version | Supported          | Notes |
| ------- | ------------------ | ----- |
| 3.x.x   | :white_check_mark: | Active development |
| 2.x.x   | :white_check_mark: | Security updates and critical bug fixes only |
| < 2.0.0 | :x:                | Unsupported |

## Reporting a Vulnerability

If you discover a security vulnerability in Manael, please do not report it through public GitHub issues.

Instead, please send an email to **contact@manael.org**.

We will review your report and respond as quickly as possible. When reporting, please include as much information as possible, such as:
* The version of Manael you are using
* Steps to reproduce the vulnerability
* Any potential impact you have identified
